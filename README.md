- 国または地域を選択

  - continue
- 文字入力および音声入力の言語
  - continue
- アクセシビリティ
  - 今はしない
- Wi-Fiネットワークを選択
  - SSID:mse_common
  - PW:m29-c0mm0n

- データとプライバシー
  - continue
- 移行アシスト
  - 今はしない

  - このタイミングでしてもしなくても良い
- 利用規約
  - 同意する
- コンピュータアカウントを作成
  - フルネーム：Kai Yoshida（日本語可）
  - アカウント名：kai（日本語不可）
  - パスワード:覚えやすいもの（4文字以上）
  - ヒント：任意
  - アイコン画像：好きな画像を設定可
	＊アカウント名は後から変更できない．
	また，Terminalで表示されるアカウント名が長くなると
	カッコよくない（個人の感想です）ので，
	短く設定しておくと良い．

  - continue
